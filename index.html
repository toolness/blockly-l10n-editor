The Blockly code above can be used to provide a localization for the example message. It can be statically or dynamically compiled into a React component for inclusion in an application.

Why Blockly?

Providing localization that feels "native", e.g. which adapts dynamically to pluralization and gender and so on, essentially seems to require writing small computer programs. Localization formats like ICU aren't particularly intuitive to developers or localizers. However, Blockly is nicely suited for allowing non-technical users to create very short programs that compile down to readable JavaScript, which makes it a potential alternative to localization message formats.
